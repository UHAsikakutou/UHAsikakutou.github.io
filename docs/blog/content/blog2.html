持っているデバイス類を紹介します

2024/3/17


お久しぶりです。気がつけば前回の投稿から1年以上経ってしまいました()
自己満で作ったサイトですが、去年の自分はまさに作成しただけで満足してしまったらしいです......
もしかしたらこの投稿も今年で最初で最後になるかもしれませんが、今回は記念すべき第１回目 (前回はテスト投稿ですからね！)ということで、持っているデバイス類を紹介します。

1. PC

まずは普通のPCから。

  • メインPC
    自作です。組んだのは前回のブログを書いた直後です。
    「GPUとCPUがボトルネックだ！」とおっしゃる方もいるかもしれませんが、安くてVRAMが多いものを選びました。
    趣味で動画編集は行いますが、AeもDαVinciも快適動作です。
    HDDの容量を無駄にしてまでRAID5にしたのは完全に趣味です()
    • CPU: Intel Core i7-13700K
    • GPU: ASUS Dual-RTX3060-012G
    • M/B: ASRock Z690 Steel Legend WIFI 6e
    • RAM: Crucial DDR4-3200 64GB kit (32GB*2)
    • SSD: Western Digital WD_BLACK SN770 1TB
    • PSU: 玄人志向 850W Gold
    • CPUファン: Owltech 360mm簡易水冷
    • ケース: Thermaltake Versa H26/W
    • HDD: TOSHIBA 2TB 7200RPM*2 + WD Blue 4TB = 4TB(RAID5)
  • MacBook Pro (13-inch,M1,2020)
    メインノートです。去年はジャンクで購入した2016年のProを使っていましたが、それを後輩に譲り、新しいのを購入しました。
    M1は前のMBPより前に機会があってAirを持っていた時期があったのですが、Intel Macを触ってから戻ってくると、改めてM1の凄さを実感しました。
    あの薄さで動画編集もモデリングもサクサクなのが本当に凄いです。さらにバッテリー持ちもいい。
    今ではM3まで出てきたAppleシリコンですが、M1はまだまだ現役です。(14インチ欲しいんですけどね...)
    デザインは旧来のウェッジシェイプ(?)です。底面が分厚く、側面に行くに従って薄くなっていくので、全体的に見ると薄く見えます。
    このデザイン、まさに「MacBookだ！」という感じがして愛していたのですが、なんでやめてしまったんですかね。。。
    最近のMacBookはのっぺりとしていて正直ダサいです()
    また、このMBPは後続のM2とともに最後のTouch Bar搭載機です...が、正直使い道はありません。ただかっこいいだけです。
    私はPockというアプリを使って、Touch BarにDockや日付や時刻を表示させています。
    • SoC: Apple M1
    • RAM: 16GB
    • SSD: 512GB
  • NEC VersaPro VG-J
    検証用サブノートです。制限を回避しWin11をインストールしています。
    4th i5(電力制限で1.5GHzぐらいしか出ない)、RAM4GB(増設不可)というわけで、実用性は皆無ですが、なんと画面がWQHDです。
    Retinaと比べると見劣りしますが、それでもWQHDはいいですね。
    画質はいいので、メインからMiracastして2画面目にすることが多いです。
    またLinuxを入れたり、Hackintoshを試したりするのに使っています。
  • 他に持っているサブノート
    他にもノートPCは大量に持っているのですが、ほとんど後輩に貸したり、分解したまま放置したりしています。
    覚えている限り箇条書きしておきます。
    • Dynabook R73
    • Latitude 5940
    • Lifebook U938
    • ThinkPad X260

2. レトロPC

次にレトロPCです。

  • PC-9821 Xa16/V13
    メインの(?)PC-98です。数年前にBEEPさんで購入しました。
    東方旧作(親戚宅から発見されたものを譲り受けた)を遊んだり、PC-98のゲーム作りに挑戦したりしています。
    はじめはXa16を購入したのですが、メモリの接触が悪いのか起動しなくなってしまったので、V13とニコイチになっています。
    Windows95は概ね動作します。(Word95は快適です！)が、GAが内臓のものなので、ゲーム系はきついと思われます。(「トロイデント」ではないんですけどね)
    • CPU: Pentium 133MHz
    • G/A: CIRRUS LOGIC GD5440 (VRAM 1MB)
    • CD-ROM: 6倍速
    • 固定ディスク: 内蔵(IDE) Maxtor 40GB (4GBに容量制限)
    • OS: Windows 95/MS-DOS 5.0A
    • 音源: 内蔵MATE-X PCM、CバスPC-9801-118
  • PC-9821 As
    誇り高きA-MATEです。V13の約半年後に購入しました。
    RAMも増設済み、CPUもODPだったのですが、動作確認後数日で起動しなくなってしまいました。分解してもそれらしき箇所は見当たらないので、原因は不明です。(動作確認時は動いてその後即動かなくなる現象、なんなんですかね)
    ひとまずコンデンサを全て交換してみようと思っています。
  • IBM Thinkpad 900
    こちらも誇り高きThinkPadです。秋葉原でジャンク品として購入したのですが、BIOSロックがかかっています。
    しかも起動画面が某社の名前とロゴ入りで、なんで流れて来たのかわかりません。
    この時代のThinkPadは比較的簡単にBIOSロック解除できると聞いたのですが、果たしてよくわかりません()
    今は分解された状態で倉庫の肥やしになっています。

3. スマホ

  • iPhone XS
    つい最近まで使っていたものです。今はPhone 13に買い替えました。
    今は自分で遊ぶためにサブとして使っています。
    やはり有機ELはいいですね。
  • iPhone 13
    現状のメインスマホです。
    まあ安心のApple製品なので、文句をつける部分はありません()
    ただ、超広角こそProにつけて、望遠を無印でも使えるようにしてくれればいいのになぁ...と思います。
  • ZTE Libero 5G III
    友人から譲り受けたものです。Apple信者ですが、Androidは自由度があって面白いです。
    SIMは入っていませんが、おもちゃとして色々いじっています。

4. その他

細かいものは色々と持っているのですが、あまり1ページに詰め込むとダイアルアップの方が苦しんでしまうので程々に...

  • Apple Magic Trackpad 3
    去年購入しました。MacBookのトラックパッドは全人類が使うべき素晴らしいものですが、それを切り離して使えるのは本当に便利です。
    他社ノートのペコペコとしたものとは違い、Appleのトラックパッドは上品で押していて気持ちが良いです。
    Windows向けにも有志がドライバを作ってくれているので、それを使っています。
  • Happy Hacking Keyboard Professional HYBRID 白
    最近購入しました。これは素晴らしいですね。
    私は古いPCのキーボードが好きなのですが、まさにそんな感じの打鍵感がします。言葉で例えるのは難しいですが、メンブレンの良いところだけ切り取ってそれを増幅した感じです。
    また配色も完璧です。まさに「キーボード」の色をしています。
    この文章もHHKBで書いているのですが、次から次へと文字が出てくるのは快感ですね。
    ...とすっかり洗脳されてしまっていますが、「今の安っぽいメンブレンは嫌いだけど、メカニカルもどれも気に入らなかった」みたいな人には是非一度試して欲しいと思います。
  • Soundcore Q45 Space
    約半年前に購入しました。最近街で見かけることも多いヘッドホンです。
    これのいいところはなんといってもコスパの良さですね。音質は最高級のヘッドホン(SONYのMX5とか)には及びませんが、外出時に聞く分には十分です。
    また、ANCもついているので、電車や飛行機で使うとき、勉強中などには重宝します。
    有線と無線の両方に対応しているので、使い勝手もいいです。
    そしてこれだけの機能がついていても、価格は1.5〜1.7万とお手頃です。

というわけで、今回は持っているデバイス類を紹介してみました。次回はいつになるかわかりませんが、また何か書こうと思います() それではまた今度！
